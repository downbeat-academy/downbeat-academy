# Cadence Design System

Welcome to the component library for **Cadence** — the design system powering [Downbeat Academy](https://downbeatacademy.com).

---

## About

`cadence-core` provides a set of accessible, composable React components built with design tokens from `cadence-tokens`. Components are styled with CSS Modules and use scoped class names to avoid collisions when consumed by host applications.

The library ships as both ESM and CommonJS, with full TypeScript declarations and a bundled CSS stylesheet.

## Installation

```bash
pnpm add cadence-core
```

Then import the stylesheet in your application entry point:

```tsx
import 'cadence-core/styles.css'
```

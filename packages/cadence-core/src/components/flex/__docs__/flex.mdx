# Flex

A layout component to enforce consistent spacing, alignment, justification, and wrapping of child elements.

## Example

<Canvas />

## Usage

```ts
import { Flex, Badge } from 'cadence-core'

const Example = () => {
  return (
    <Flex
      direction='row'
      gap='medium'
    >
      <Badge text='Item 1' type='success' style='outline' />
      <Badge text='Item 2' type='error' style='fill' />
      <Badge text='Item 3' type='warning' style='light' />
    </Flex>
  );
};

export default Example;
```

## Arguments

- **direction** - The direction of the Flex container. Can be one of `row`, `column`.
- **gap** - The spacing between child elements. Can be one of `none`, `2x-small`, `x-small`, `small`, `medium`, `large`, `x-large`, `2x-large`, `3x-large`.
- **padding** - The padding of the container, can be set to `none`, `x-small`, `small`, `medium`, `large`, `x-large`, `2x-large`, `3x-large`.
- **alignItems** - The alignment of child elements. Can be one of `start`, `center`, `end`, `stretch`.
- **alignContent** - The alignment of child elements when they wrap. Can be one of `start`, `center`, `end`, `space-between`, `space-around`.
- **justifyItems** - The justification of child elements. Can be one of `start`, `center`, `end`, `space-between`, `space-around`.
- **justifyContent** - The justification of child elements when they wrap. Can be one of `start`, `center`, `end`, `space-between`, `space-around`.
- **background** - The background of the container, can be set to `primary`, `faint`, `high-contrast`, `brand, `interactive`, `success`, `warning`, or `critical`.
- **wrap** - Whether child elements should wrap to the next line. Can be one of `wrap`, `nowrap`.
- **className** - An optional class name to apply to the Flex container.
